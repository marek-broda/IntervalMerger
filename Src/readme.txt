- Instructions 
--------------

- Place CSV that you wish to process in the "CSVs" folder. 
- Run the console application with the name of the file (including .csv) 
  as the first argument, and the desired merge distance as the second 
  argument. 
- Should run through the CSV - merging, removing and deleting invervals 
  as required and writing the result to the console. 
- Stopwatch is being run on the main() function and gives a time at the end.
